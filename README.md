# Hair Diamond Beauty

Statische website voor Hair Diamond Beauty. De site bestaat uit pure HTML, CSS en vanilla JavaScript zodat hij zonder build-stap gehost kan worden (bijvoorbeeld via GitHub Pages).

## Structuur
- `index.html` – hoofdwebsite
- `privacy.html` – privacybeleid
- `assets/css/main.css` – styling
- `assets/js/main.js` – Setmore-pop-up en algemene scripts
- `images/` – alle gebruikte assets

## Ontwikkeling
Open `index.html` in je browser of host de map op eender welke statische hostingdienst. Alle boekingsknoppen openen de Setmore-agenda (`https://diamondhairbeauty.setmore.com/layla`).
